使用约定Usage Notes

ocbf -h 显示常用命令；ocbf -hh 显示完整命令。ocbf -h shows the common commands, while ocbf -hh shows the full command list.

train / relax / efs-distri / predict-xyz / plot-errors 支持 set 子用法，默认值持久化到 ~/.ocbf/cli_defaults.json。train / relax / efs-distri / predict-xyz / plot-errors support the set sub-usage, and persist defaults in ~/.ocbf/cli_defaults.json.

默认值按当前代码逻辑整理；若某字段没有稳定默认值，就在文档中记为 auto / 无。Defaults are taken from the current code. If a field has no stable built-in default, it is marked as auto / none.
